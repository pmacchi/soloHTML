4- ¿Qué utilidad tiene el sitio GitHub?
GitHub sirve para subir repositorios locales a la nube de GitHub mismo. Lo utilizamos para subir los repositorios, y en caso de trabajar con mas de una persona, ir pusheando los cambios que se van haciendo a lo largo del proyecto.
5- ¿Qué es Bootstrap? y Flexbox?
Bootstrap es framework de CSS listo para usar en el diseño de las paginas. Flexbox es un modelo de diseño en CSS para alinear y distribuir los elementos de manera flexible.
6- Describe brevemente diferencias entre Git y GitHub
Las diferencias entre GIT y GitHub es que GIT es una herramienta de control de versiones, no necesita conexion a Internet para la mayoria de las operaciones y se utiliza de manera local, mientras que GitHub es uan herramienta de hospedaje de codigo en la nube, necesita conexion a internet y se utiliza para facilitar la colaboracion en proyectos.
