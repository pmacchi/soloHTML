6- Describe brevemente diferencias entre Git y GitHub
Las diferencias entre GIT y GitHub es que GIT es una herramienta de control de versiones, no necesita conexion a Internet para la mayoria de las operaciones y se utiliza de manera local, mientras que GitHub es uan herramienta de hospedaje de codigo en la nube, necesita conexion a internet y se utiliza para facilitar la colaboracion en proyectos.

Comandos Git para Subir Cambios

1. Cambiar a la rama principal

git checkout main

Este comando cambia a la rama principal (main) de tu repositorio.

2. Agregar los cambios al área de preparación

git add .

Este comando agrega todos los archivos modificados al área de preparación.

3. Crear un commit con una descripción de los cambios

git commit -m "Descripción de los cambios"

Este comando crea un commit con los cambios que has agregado al área de preparación.

4. Subir los cambios al repositorio remoto

git push origin main

Este comando sube tus commits desde tu repositorio local al repositorio remoto en GitHub.
